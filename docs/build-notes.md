# 제작 노트: 국내증시 수급분석기, 5일짜리 표에서 120거래일 전략 대시보드까지

## 1. 시작점

처음 목표는 간단했습니다.

네이버 증권에서 국내 종목을 검색하고, 외국인·기관·개인 수급을 보기 좋게 정리하는 Chrome 확장프로그램을 만드는 것이었습니다.

하지만 첫 버전은 최근 5거래일 중심이었습니다.  
데이터는 보였지만 실제 판단에는 부족했습니다.

## 2. 문제

수급은 하루나 이틀의 숫자보다 누적 흐름이 중요합니다.

국내증시는 외국인, 기관, 개인이라는 투자 주체별 매매 동향을 확인할 수 있습니다. 가격이 결과라면 수급은 그 결과를 만든 돈의 흐름이기 때문에, 누가 사고 누가 빠지는지를 함께 봐야 가격 움직임을 더 입체적으로 해석할 수 있습니다.

예를 들어 다음 질문은 5일치 데이터만으로는 답하기 어렵습니다.

- 외국인과 기관이 계속 빠지고 있는가?
- 가격 하락 중에도 누가 받아내고 있는가?
- 가격 상승 중 주체 수급이 이탈하고 있는가?
- 외국인 보유율이 추세적으로 변하고 있는가?

그래서 단순 표가 아니라 장기 분석 구조가 필요했습니다.

## 3. 데이터 수집 방식

모바일 네이버 증권 API는 최근 실제 개인 수급을 얻는 데 유용했습니다.

```text
https://m.stock.naver.com/api/stock/{종목코드}/integration
```

하지만 이 API는 최근 5거래일 중심입니다.

장기 수급은 PC 네이버 금융의 외국인/기관 페이지를 사용했습니다.

```text
https://finance.naver.com/item/frgn.naver?code={종목코드}&page={페이지}
```

이 페이지는 1페이지당 약 20거래일의 데이터를 제공합니다.

최종 구조는 다음과 같습니다.

- 모바일 API: 최근 5거래일 실제 개인 수급
- PC 수급 페이지: 장기 외국인·기관 수급
- 개인/기타: 외국인·기관 값을 기준으로 계산한 잔여 추정치

## 4. 분석 지표

단순히 값을 보여주는 대신 다음 지표를 만들었습니다.

- 외국인+기관 누적 수급
- 개인/기타 누적 수급
- 외국인 보유율 변화
- 가격 변화율
- 거래량 대비 수급강도
- 가격과 수급의 동행률
- 최대 매집일
- 최대 이탈일
- 개인/기타와 외국인+기관의 반대 움직임 횟수

## 5. 전략화

최종 목표는 데이터 표가 아니라 판단 보조였습니다.

그래서 수급 점수를 만들고, 다음 다섯 단계로 분류했습니다.

- 매수 우위
- 조건부 매수
- 관망
- 매도 우위
- 강한 매도 우위

각 분류에는 실행 전략을 붙였습니다.

- 분할매수
- 소액 분할
- 대기
- 비중축소
- 신규매수 금지

## 6. 시각화

숫자만 나열하면 흐름이 잘 보이지 않습니다.

그래서 팝업을 760px 대형 대시보드로 키우고 SVG 차트를 넣었습니다.

- 수급 막대 그래프
- 가격 라인 그래프
- 외국인 보유율 라인 그래프

이렇게 바꾸니 “누가 사고 있는가”와 “가격이 어떻게 반응하는가”를 한 화면에서 볼 수 있게 됐습니다.

## 7. 배운 점

이번 프로젝트에서 가장 중요한 포인트는 데이터 수집이 아니라 데이터 해석 구조였습니다.

표만 있으면 확인은 가능하지만 판단은 어렵습니다.

이번 프로젝트는 다음 순서로 발전했습니다.

```text
데이터 수집 → 표 → 그래프 → 점수 → 전략
```

브라우저 확장프로그램은 단순 편의 기능을 넘어, 웹페이지 데이터를 다른 판단 도구로 재구성할 수 있다는 점이 흥미로웠습니다.
